# NoFakes-Code-Challenge

## Description

NoFakes Code Challenge with [Nest](https://github.com/nestjs/nest).

## Installation

```bash
$ npm install
```

```bash
# run once to install husky if you are going to commit something 
$ npm prepare
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
